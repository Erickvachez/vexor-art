## Gallery folders

Put product images into the matching folder below.

### Recommended naming

- Use simple names like `1.jpg`, `2.jpg`, `3.jpg`, `4.jpg`
- Or use descriptive names like `ocean-drift.jpg`

### Folders

For each product:

- `{product}/slideshow/` – images used in the top slider
- `{product}/catalog/` – images used in the ready-made stock grid

Products:

- `chessboards/`
- `herb-grinders/`
- `broken-bottle-wall-art/`
- `custom-wall-art/`
- `letter-keychains/`
- `trays/`
